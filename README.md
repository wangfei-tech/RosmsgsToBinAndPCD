# RosmsgsToBinAndPCD
将rosbag中的点云信息转换成pcd和bin的文件形式

# Dependencies

ubuntu 20.04

ros noetic

pcl-1.10

boost1.7.1

# Use

`cd ~/`

`mkdir catkin_ws/src`

`cd catkin_ws/src`

`git clone https://github.com/wangfei-tech/RosmsgsToBinAndPCD.git`

`cd ../`

`catkin_make`

`source dev/set.bashrc`

`rosrun *** ****`
